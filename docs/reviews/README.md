# Review-Protokolle — Momentaufnahmen

Alle Dokumente in diesem Ordner sind **historische Momentaufnahmen**: Sie beschreiben den
Projektstand **zum jeweiligen Datum** und werden nachträglich nicht umgeschrieben. Dort
genannte Befunde, Testzahlen und offene Punkte sind **keine aktuellen Anforderungen** —
vieles ist längst abgearbeitet.

Wo der aktuelle Stand steht:
- **Erledigungsstatus je Review:** `docs/ideen-backlog.md` (Bewertungs-Abschnitte) bzw. die
  „Fortschritt"-Blöcke in `docs/m2-plan.md`.
- **Aktuelle Prüfstände (Tests/Lint/Build):** die CI auf `main`.
